# Лабораторная работа №2

Форма сдачи лабораторной работы:
- текстовый документ с определенными параметрами реального процесса, указанием использованных команд и скриншотами;
- файл скрипта;
- демонстрация работы скрипта.

## Задание №1

Выбрать любой реальный процесс системы и определить для него:
- PID процесса;
- PPID;
- текущее состояние;
- использование памяти;
- список открытых файлов;
- дочерние процессы (если есть).

## Задание №2

Написать скрипт, который:
- запускает 3 фоновых процесса (sleep с разным временем);
- сохраняет их PID;
- ожидает завершения каждого через `wait`;
- выводит код завершения каждого;
- завершается только после завершения всех дочерних процессов.
