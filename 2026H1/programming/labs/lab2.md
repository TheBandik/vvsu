# Лабораторная работа №2

## Задание 1 – Карточка товара

Создайте переменные:
- название товара (строка);
- цена (вещественное число);
- количество на складе (целое число);
- товар в наличии (логическое значение).

Используя `f-строки`, выведите информацию в формате:

```
Товар: ...
Цена: ...
Количество на складе: ...
В наличии: ...
```

## Задание 2 – Финансовая сводка

Разбейте предложение: `Компания получила прибыль в текущем квартале` на несколько переменных (по 1–2 слова).

Соберите предложение обратно с помощью конкатенации строк и выведите его.

## Задание 3 – Рекламный баннер

Создайте строку: `Акция! Скидки!`, которая повторяется 4 раза подряд, используя умножение строки.

## Задание 4 – Средняя прибыль

Пользователь вводит через запятую:

```
доход,расход,количество_месяцев
```

Необходимо:
- сохранить значения в три переменные;
- вычислить прибыль (доход − расход);
- найти среднюю прибыль в месяц (целочисленное деление);
- вывести: `Средняя прибыль в месяц: <значение>`.

## Задание 5 – Анализ кода контракта

Пользователь вводит номер договора (строку длиной не менее 10 символов).

С помощью срезов вывести:
- первые 4 символа (код организации);
- последние 2 символа (контрольный код);
- символы с 4 по 8;
- номер в обратном порядке.
