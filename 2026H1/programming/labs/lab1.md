# Лабораторная работа №1

## Задание №1 – Типы данных

Объявите переменные следующих типов:
- целое число (количество товаров)
- вещественное число (цена товара)
- строка (название товара)
- логическое значение (товар в наличии)

Выведите все значения в консоль.

## Задание 2 – Информация о пользователе

Запросите у пользователя:
- имя 
- возраст

Выведите в консоль строку: `Пользователь: <имя>, возраст: <возраст>`

## Задание 3 – Простая покупка

Пользователь вводит:
- цену товара
- количество

Вычислите итоговую стоимость покупки и выведите ее в консоль.

## Задание 4 – Скидка

Пользователь вводит:
- цену товара
- процент скидки

Найдите итоговую цену после скидки, сумму скидки и выведите полученные результаты в консоль.

## Задание 5 – Зарплата сотрудника

Пользователь вводит:
- оклад
- премию
- процент налога

Вычислите и выведите в консоль:
- зарплату до налога
- сумму налога
- зарплату к выплате

## Задание 6 – Прямоугольное помещение

Пользователь вводит:
- длину
- ширину

Вычислите и выведите в консоль:
- площадь
- периметр

## Задание 7 – Мини-чек

Пользователь вводит:
- название товара
- цену
- количество

Выведите чек:

```
Товар: ...
Цена: ...
Количество: ...

Итого: ...
```

## Задание №8 – Индекс продуктивности дня

Пользователь вводит:
- количество часов сна
- количество часов работы
- количество часов отдыха

Вычислите и выведите в консоль коэффициент продуктивности по формуле: `(часы работы * 2 + часы отдыха) / часы сна`

## Задание №9 – Генератор титулов

Пользователь вводит:
- имя
- эпитет (например: великий, имбовый, дикий)
- локацию (например: интернета, общаги, вселенной)

Вывести сообщение: `Встречайте: <имя> — <эпитет> босс <локация>`

## Задание №10 – Рисунок

Вывести в консоль мордочку кота из символов:

```
 /\_/\ 
( o o )
  >^<
```
