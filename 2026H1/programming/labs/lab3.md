# Лабораторная работа №3

## Задание 1. Подсчёт выручки за несколько дней

Магазин работает несколько дней.
Каждый день вводится сумма выручки.

Программа должна:

1. Запросить количество дней.
2. В цикле запросить выручку за каждый день.
3. Посчитать **общую выручку**.

Вывести:

```
Общая выручка: ...
```

## Задание 2. Средняя дневная выручка

Магазин работает `N` дней.

Программа должна:

1. Запросить количество дней.
2. В цикле ввести выручку за каждый день.
3. Найти **среднюю выручку**.

Вывести:

```
Средняя выручка: ...
```

## Задание 3. Общий объём продаж товара

В магазине продаётся один товар.

Программа должна:

1. Запросить **цену товара**.
2. Запросить **количество дней продаж**.
3. В цикле вводить **сколько товаров продано за день**.
4. Посчитать **общую выручку**.

## Задание 4. Подсчёт прибыли

Компания каждый день получает доход и несёт расходы.

Программа должна:

1. Запросить количество дней.
2. В цикле вводить:
    * доход
    * расходы
3. Посчитать **общую прибыль**.

Формула:

```
прибыль = доход - расходы
```

## Задание 5. Сумма инвестиций

Инвестор вкладывает деньги в разные проекты.

Программа должна:

1. Запросить количество проектов.
2. В цикле вводить сумму инвестиций в каждый проект.
3. Посчитать **общую сумму инвестиций**.

## Задание 6. Повторение строки (экономический отчёт)

Программа должна:

1. Запросить название компании.
2. Запросить количество строк отчёта.
3. В цикле вывести строку, указанное количество раз:

```
Компания <название> ведёт финансовый учёт.
```
