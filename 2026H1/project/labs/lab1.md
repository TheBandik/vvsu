# Лабораторная работа №1

Все лабораторные работы выполняются по единожды выбранной предметной области: [варианты предметных областей](vars.md)

## Задание №1

Провести анализ предметной области и выделить:

- Цель и задачи проекта:
  - Основная цель (одно предложение)
  - Основные задачи (список из нескольких пунктов)
- Основные требования
  - Основная функциональность проекта (обязательная минимальная функциональность, без подробностей)
  - Дополнительные требования и ограничения (например: дизайн, платформа)
- Целевая аудитория проекта
- Основные хранимые сущности

## Результаты

Необходимо создать репозиторий на GitHub (или ином другом хостинге) для загрузки выполненных заданий.

Результаты оформить в текстовом формате в свободном оформлении и выгрузить в свой репозиторий.
