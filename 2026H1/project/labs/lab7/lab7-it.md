# Лабораторная работа №7 для БИН/БИС

Спроектировать логическую модель базы данных на основе требований и предметной области с учётом нормализации (3НФ) и целостности данных.

Работа выполняется на основе:
* предметной области + DFD;
* TO-BE модели;
* функциональных требований.

## Ход работы

**По каждом шагу важно дать обоснование согласно прошлым лабораторным работам**

1. Выделить основные хранимые сущности системы.
2. Для каждой сущности определить атрибуты, выделить ключевые, указать типы данных абстрактно.
3. Определить связи между сущностями, промежуточные таблицы при необходимости.
4. Определить следующие ограничения и целостность: PK / FK, уникальные поля, обязательные поля (nullable).

## Результат

* Составить ER-диаграмму, сохранить ее в PDF.
* Составить текстовое описание проведенного проектирования БД.
