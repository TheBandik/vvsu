# Лабораторная работа №6

## Задание №1

В системе обработки данных используется **стек** для анализа выражений.

Стек поддерживает следующие операции:

* добавление элемента в стек (push)
* удаление элемента из стека (pop)
* просмотр верхнего элемента (top)
* проверка на пустоту

Система получает строку, состоящую из круглых скобок `(` и `)`.

Необходимо определить, является ли последовательность скобок корректной.

Последовательность считается корректной, если:

* каждая открывающая скобка имеет соответствующую закрывающую
* порядок скобок соблюдён

---

**Требования:**

* реализовать стек в виде класса
* использовать собственную реализацию (без `list.pop()` как «готового стека» – можно хранить массив, но логика должна быть явно описана)
* реализовать алгоритм проверки через стек

---

### Формат входных данных

В единственной строке задана строка `s` (`1 ≤ |s| ≤ 10^5`), состоящая только из символов `(` и `)`.

---

### Формат выходных данных

Выведите:

* `YES`, если последовательность корректна
* `NO` в противном случае

---

### Тесты

1 input:

```bash id="st1"
(())
```

1 output:

```bash id="st2"
YES
```

---

2 input:

```bash id="st3"
(()()
```

2 output:

```bash id="st4"
NO
```

---

3 input:

```bash id="st5"
)(
```

3 output:

```bash id="st6"
NO
```

---

## Задание №2

В системе хранения данных используется хэш-таблица с разрешением коллизий методом цепочек.

Таблица хранит пары вида:
`ключ (идентификатор студента) – значение (оценка)`.

Вам необходимо реализовать класс `HashTable`.

---

**Требования:**

* реализовать класс с методами:

    * `__init__(self, size=11)`
    * `_hash(self, key)` – хэш-функция по методу остатков (`key % size`)
    * `insert(self, key, value)`
    * `search(self, key)`
    * `delete(self, key)`
    * `display(self)`
* использовать метод цепочек (списки)
* при вставке:
    * если ключ уже существует – обновить значение
* при удалении:
    * удалить только один элемент
    * вернуть `True`, если удаление произошло, иначе `False`

---

### Формат входных данных

В первой строке задано число `n` (`1 ≤ n ≤ 1000`) – количество операций вставки.

В следующих `n` строках заданы пары:
`key value`

В следующей строке задан ключ `x` – существующий.

В следующей строке задан ключ `y` – несуществующий.

В последней строке задан ключ `d` – ключ для удаления.

---

### Формат выходных данных

Выведите:

1. значение по ключу `x`
2. значение по ключу `y` (`None`, если отсутствует)
3. результат удаления (`True` или `False`)
4. содержимое таблицы после удаления

---

### Тесты

1 input:

```bash id="ht1"

105 2

102
```

1 output:

```bash id="ht2"

None
True
[[], [(101, 5)], [], [(103, 3)], [(104, 5)], [(105, 2)], [], [], [], [], []]
```

---

2 input:

```bash id="ht3"

12 20
23 30
12
5

```

2 output:

```bash id="ht4"

False
```
