# Лабораторная работа №1

## Задания

### Вариант №1 – «Калькулятор расходов»

Реализуйте программу на Python, которая позволяет вести учёт личных расходов.

Программа должна:
- хранить расходы в словаре, где ключ – категория расхода, например «Продукты», «Транспорт», «Развлечения», а значение – список сумм расходов по этой категории;
- предоставлять пользователю меню с действиями: добавить расход, показать все расходы, показать статистику и выход;
- программа должна работать в цикле до тех пор, пока пользователь не выберет «Выход»;
- предусмотрите обработку некорректного ввода, например, если пользователь вводит текст вместо числа для суммы.

### Вариант №2 – «Сортировка списка книг»

Реализуйте программу на Python, которая работает со списком книг. Каждая книга представлена словарём с полями: title, author, year, pages.

Программа должна:
- создать список из не менее 10 книг;
- вывести весь список книг в удобочитаемом формате, например, нумерованный список;
- реализовать сортировку списка книг по выбору пользователя;
- после сортировки вывести отсортированный список;
- использовать встроенную функцию sorted() или метод sort() с параметром key.
