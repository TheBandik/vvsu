# Лабораторная работа №3

## Задание №1

Написать программу, которая:
- создаёт переменные разных типов и выводит для каждой: значение, тип, id() и размер через sys.getsizeof();
- демонстрирует, что две переменные могут указывать на один объект (оператор is);
- показывает разницу в поведении при присваивании для изменяемых и неизменяемых типов.

Оцените асимптотическую временную сложность написанного вами кода, обоснуйте ответ.

## Задание №2

Дан массив чисел (сгенерируйте). Найти максимальный элемент, используя четыре подхода:
- Brute force – сравнить каждый элемент с каждым;
- Greedy – пройти массив один раз, обновляя текущий максимум;
- Divide & Conquer – разделить массив пополам, найти максимум в каждой половине рекурсивно;
- DP – использовать табуляцию: dp[i] = max(dp[i-1], arr[i]).

Для каждого подхода укажите сложность и замерить время на массиве из n=100000 элементов.

Сделайте информативный вывод в консоль с фиксацией результатов замера времени и сложности по каждому подходу.
