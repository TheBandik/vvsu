# Лабораторная работа №4

## Задание №1

В системе обработки данных используется простой механизм «самоупорядочивания» списка задач.

Список задач представлен массивом чисел – чем меньше число, тем выше приоритет задачи. Система работает следующим образом:

* на каждом цикле она проходит по списку слева направо
* сравнивает соседние задачи
* если задача с меньшим приоритетом стоит перед задачей с большим приоритетом, они меняются местами

Циклы повторяются до тех пор, пока за один полный проход не произойдёт ни одного изменения.

Ваша задача – определить, сколько таких циклов потребуется системе, чтобы полностью упорядочить список задач.

### Формат входных данных

В первой строке задано число `n` (`1 ≤ n ≤ 5000`) – количество задач.

Во второй строке задано `n` целых чисел – приоритеты задач (`|a_i| ≤ 10^9`).

### Формат выходных данных

Выведите одно число – количество циклов обработки.

### Тесты

1 input:

```bash id="l5v84e"
5
1 2 3 4 5
```

1 output:

```bash id="l3xvsm"
1
```

2 input:

```bash id="e4yxk7"
5
5 4 3 2 1
```

2 output:

```bash id="b0o0j3"
5
```

3 input:

```bash id="bn8u2n"
6
1 5 2 3 4 6
```

3 output:

```bash id="b6z6gr"
2
```

## Задание №2

В системе хранения данных используется отсортированный список идентификаторов.

Для поиска элемента применяются два различных режима:

1. **Последовательный режим** – система проверяет элементы один за другим, начиная с первого
2. **Интеллектуальный режим** – система проверяет элемент в середине текущего диапазона и, в зависимости от результата, продолжает поиск только в одной из половин

Вам необходимо оценить эффективность работы системы.

Для заданного массива и значения `x` определите, сколько проверок потребуется в худшем случае для каждого режима поиска.

Если элемент отсутствует, система завершает поиск только тогда, когда это становится очевидно.

### Формат входных данных

В первой строке задано число `n` (`1 ≤ n ≤ 100000`).

Во второй строке задан отсортированный массив из `n` целых чисел.

В третьей строке задано число `x`.

### Формат выходных данных

Выведите два числа через пробел:

* количество проверок в последовательном режиме
* количество проверок в интеллектуальном режиме

### Тесты

1 input:

```bash id="gh6n7m"
5
1 3 5 7 9
7
```

1 output:

```bash id="0xq9q2"
4 3
```

2 input:

```bash id="s02g7v"
5
1 3 5 7 9
2
```

2 output:

```bash id="sy0yye"
5 3
```
