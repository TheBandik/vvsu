# Лабораторная работа №5

## Задание №1

В системе обработки данных используется односвязный список для хранения последовательности чисел.

Список поддерживает следующие операции:

* добавление элемента в начало списка
* вывод всех элементов списка
* определение длины списка
* поиск элемента по значению

Вам необходимо реализовать модель работы такой системы и выполнить заданные операции.

После формирования списка требуется:

* вывести его содержимое
* определить длину списка
* найти позицию заданного элемента

Если элемент отсутствует, система возвращает значение `-1`.

**Односвязный список реализовать в виде класса с методами. Должны быть собственные реализации алгоритмов в методах. Использовать готовые функции Python запрещено.**

### Формат входных данных

В первой строке задано число `n` (`0 ≤ n ≤ 10^5`) – количество элементов, добавляемых в список.

Во второй строке задано `n` целых чисел – элементы списка (в порядке их добавления в начало).

В третьей строке задано число `x` – элемент, который необходимо найти.

В четвёртой строке задано число `y` – элемент, который гарантированно отсутствует в списке.

### Формат выходных данных

Выведите:

1. Содержимое списка в формате
   `a → b → c → None`
2. Длину списка
3. Позицию элемента `x`
4. Позицию элемента `y`

Каждое значение выводится с новой строки.

### Тесты

1 input:

```bash id="ll1"
5
10 20 30 40 50
15
99
```

1 output:

```bash id="ll2"
50 → 40 → 30 → 20 → 10 → None
5
-1
-1
```

2 input:

```bash id="ll3"
4
5 10 15 20
15
99
```

2 output:

```bash id="ll4"
20 → 15 → 10 → 5 → None
4
1
-1
```

## Задание №2

Система хранения данных была расширена и теперь поддерживает дополнительные операции над односвязным списком:

* добавление элемента в конец списка
* вставка элемента на заданную позицию
* удаление элемента по значению
* преобразование списка в массив

Правила работы:

* если позиция вставки больше длины списка – элемент добавляется в конец
* если позиция меньше 0 – возникает ошибка
* при удалении удаляется **первое вхождение** элемента

Вам необходимо реализовать модель работы системы и выполнить заданные операции.

### Формат входных данных

В первой строке задано число `n` (`0 ≤ n ≤ 10^5`) – количество элементов списка.

Во второй строке задано `n` целых чисел.

В третьей строке заданы два числа `pos` и `val` – позиция и значение для вставки.

В четвёртой строке задано число `del` – значение для удаления.

### Формат выходных данных

Выведите:

1. Список после вставки
2. Результат удаления (`True` или `False`)
3. Итоговый список
4. Представление списка в виде массива

### Тесты

1 input:

```bash id="ll5"
3
1 2 3
1 99
2
```

1 output:

```bash id="ll6"
1 → 99 → 2 → 3 → None
True
1 → 99 → 3 → None
[1, 99, 3]
```

2 input:

```bash id="ll7"
2
10 20
5 99
15
```

2 output:

```bash id="ll8"
10 → 20 → 99 → None
False
10 → 20 → 99 → None
[10, 20, 99]
```
