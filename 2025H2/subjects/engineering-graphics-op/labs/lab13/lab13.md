# Лабораторная работа №13

## Задание

### Исходные данные для первого маршрута

Логистический маршрут проходит через следующие объекты:

- **Склад A** — отправная точка (зона «Город»)
- **Транспортный узел T1** — стык автомобильного направления
- **Распределительный центр B** — начало железнодорожного плеча (зона «Пригород»)
- **Транспортный узел T2** — стык ж/д и автомобильного транспорта
- **Перевалочная база C** — зона «Индустрия»
- **Порт D** — конечная точка, портовый район

#### Расстояния и типы транспорта по участкам

| Маршрут | Тип транспорта    | Расстояние |
|---------|-------------------|------------|
| A → T1  | Авто              | 12 км      |
| T1 → B  | Авто              | 20 км      |
| B → T2  | Ж/Д               | 84 км      |
| T2 → C  | Авто              | 15 км      |
| C → D   | Морское сообщение | 6 км       |

### Исходные данные для второго маршрута

Логистический маршрут проходит через следующие объекты:

- **C0** — Центральный склад (городская зона)
- **N1** — Узел городских магистралей
- **H2** — Хаб перераспределения (железнодорожная станция)
- **N3** — Универсальный транспортный узел
- **W4** — Склад W4 (индустриальный склад)
- **E5** — Экспортный терминал (портовая зона)

#### Расстояния и типы транспорта по участкам

| Маршрут | Тип транспорта    | Расстояние |
|---------|-------------------|------------|
| C0 → N1 | Авто              | 10 км      |
| N1 → H2 | Авто              | 18 км      |
| H2 → N3 | Ж/Д               | 62 км      |
| N3 → W4 | Авто              | 14 км      |
| W4 → E5 | Морское сообщение | 9 км       |

### Построение схем маршрута

1. Создайте новый документ в режиме **Чертеж**
2. Добавьте два листа, в настройках оформления листов выберете **Без основной надписи**, формат листа **А3
   горизонтальная ориентация**
3. Разместите условные графические обозначения логистических объектов вместе с легендой:
    - Склады – прямоугольники с наименованием склада внутри
    - Транспортные узлы – окружности
    - Порты – трапеции
    - Границ зон – штрих-пунктир
4. Соедините точки маршрута линиями различных типов вместе с легендой:
    - Авто – сплошная тонкая линия
    - Ж/Д – двойная сплошная основная линия
    - Морское сообщение – штриховая линия
5. Рядом с каждым участком проставьте расстояние
