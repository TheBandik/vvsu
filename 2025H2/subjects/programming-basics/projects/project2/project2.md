# Проект №2: Трекер доходов и расходов

Создайте консольное приложение, которое позволяет пользователю вводить данные о доходах и расходах, распределяя их по категориям. Программа должна предоставлять итоговый баланс, возможность просмотра данных за определённый период и анализ расходов.

## Требования к проекту

### Добавление записей

- Пользователь может вводить сумму, категорию, дату и описание для доходов и расходов.

### Категории

- Предустановленные категории (например, "Еда", "Транспорт", "Развлечения").
- Возможность добавления пользовательских категорий.

### Просмотр данных

- Итоговый баланс за всё время.
- Список всех записей с фильтром по дате и категории.

### Анализ

- Подсчёт расходов по категориям.
- Топ категории расходов.
- Соотношение доходов и расходов.

Можно реализовать дополнительную функциональность приложения по желанию.

## Результаты
