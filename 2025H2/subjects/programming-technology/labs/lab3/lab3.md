# Лабораторная работа №3

## Задание №1

Необходимо провести **два** сравнительных анализа инструментов разработки, потенциально применимых в вашем проекте. 

1. Выберите **минимум три** инструмента для сравнения, подходящих для реализации нужной части проекта.
2. Определите критерии сравнения (например: функциональность, удобство использования, кроссплатформенность, стоимость, документация и сообщество, совместимость с другими технологиями и т.д.).
3. Сравните инструменты в виде **таблицы**, указав их **достоинства и недостатки**.
4. Сделайте краткий текстовый вывод по выбранному инструменту.

Результат выполнения задания оформить в виде текстового документа и сдать преподавателю. Загружать в репозиторий и добавлять в техническое задание необязательно, можно по желанию.

## Задание №2

На основании анализа и концепции проекта необходимо определить технологический стек, который будет использоваться при проектировании и разработке.

Оформление документа свободное. Документ необходимо загрузить в репозиторий проекта. В техническое задание стек добавлять необязательно, можно по желанию.
